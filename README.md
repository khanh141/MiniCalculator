# MiniCalculator

Một ứng dụng máy tính đơn giản trên web được xây dựng bằng Streamlit. 
Ứng dụng hỗ trợ các phép toán cơ bản như cộng, trừ, nhân và chia. 
Ứng dụng xử lý việc thay thế toán tử, kiểm tra số thập phân và xóa dữ liệu nhập sau khi tính toán.
